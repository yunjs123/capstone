# AtmoDust
This is our Fall 2020 UNCC Capstone project called AtmoDust. We are using public air quality data provided by the EPA to create an interactive map that shows air quality data for an area as well as make predictions on future air quality conditions.
## Starting the Server
Once you have downloaded this repo, make sure you have python and flask installed. Then issue the following commands:
```shell
flask run --host=0.0.0.0 --port=8080
```

## Locations for Files
- HTML files go in app/templates/ folder
- CSS files go in app/static/styles/ folder
- SVG files go in app/templates/ folder
